# Формирование выборки дерева
1. Есть flyway скрипт для создания view, используемого для выборки
2. Создан rest сервис для формирования дерева с использованием Open API (подход API first)
   Вызов из swagger
   /swagger-ui/index.html#
 
   GET
   /api/plm/plmTree
   Иерархическое дерево

3. Есть не используемый раздел LOGIN (для работы с авторизацией).
4. Пример выгрузки дерева приведен в файле response_1733409786517.json
   Для демонстрации и отладки я изменил ссылки на parent в таблице plmhierarchy.
   Измененные данные в файле plmhierarchy_new.csv.
5. Возможно, для отображения в виде дерева достаточно было бы просто вернуть список из view.
   Но этол зависит от компонента, используемого на фронте для отображения.
